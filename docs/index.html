LaTeX Template

Standard template for clean, professional proof write-ups:

\documentclass[11pt]{article}
\usepackage{amsmath,amssymb,amsthm}
\usepackage[margin=1in]{geometry}
\newtheorem*{claim}{Claim}

\begin{document}
\section*{Putnam Practice \#N \quad [Your Name]}

\subsection*{Problem A1}
\begin{claim}
% State the result you will prove clearly
\end{claim}

\textbf{Key Idea:} % One sentence describing the main approach

\textbf{Proof:} 
% Step-by-step argument with clear justification
% Use proper mathematical notation
% Address edge cases if relevant

\qed
\end{document}
