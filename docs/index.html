Proof Playbook

Eight moves that crack most A1–A3 / B1–B2 problems. Learn the patterns, then write them clearly.

Induction & invariants

  • Strong/double induction on structure
  • Monovariants

Inequalities

  • AM‑GM, Cauchy‑Schwarz/Titu’s
  • Jensen/convexity, rearrangement

Number theory

  • Parity, modular arithmetic
  • Pigeonhole, orders (lite)

Combinatorics

  • Extremal, invariants
  • Double counting, injections

Series/analysis

  • Telescoping
  • Integral bounds & estimates

Algebra/polynomials

  • Vieta, factor patterns
  • Root bounds, Newton sums (lite)

Linear algebra ideas

  • Rank‑nullity insight
  • Trace/det invariants, PSD via A Aᵀ

Functional sanity

  • Plug easy points
  • Monotonicity/injectivity checks

Write‑up Standard

State the claim → one‑line idea → proof steps → edge cases → boxed result. Clean partials get credit.
